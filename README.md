# RHF-MUI-KIT

React Hook Form MUI Component Library

## Installation

```bash
npm install @chris-c-brine/rhf-mui-kit
```


## Development

### Setup

```bash
git clone https://github.com/chris-c-brine/rhf-mui-kit.git
cd rhf-mui-kit
npm install
```

### Build

```bash
npm run build
```

### Development Server

```bash
npm run dev
```
